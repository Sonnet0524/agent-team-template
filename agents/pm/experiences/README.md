## 使用说明

本目录用于存放 PM Agent 的经验总结：
- 遇到的问题和解决方案
- 有效的做法
- 无效的做法
- 改进建议

**命名规范**：
- `{任务类型}-{YYYYMMDD}.md`
- 示例: `sprint-management-20260308.md`

**记录时机**：
- 完成重要任务后
- 遇到重大问题时
- 发现更好的工作方法时
- 用户做出重要决策时
